# Projeto To-Do App

Este é um protótipo de alta fidelidade de um aplicativo de gerenciamento de tarefas, desenvolvido com HTML5, CSS3 e JavaScript puro.

## Estrutura do Projeto

```
/projeto-app
├── index.html                # Boas-vindas / Login
├── style.css                 # Estilos globais (Flexbox/Grid + Variáveis CSS)
├── script.js                 # Lógica de navegação e interatividade
├── /pages                    # Telas do aplicativo
│   ├── lista-de-tarefas.html
│   ├── nova-tarefa.html
│   └── detalhes-tarefa.html
├── /assets
│   ├── /images              # Ativos visuais
│   └── /icons               # Ícones SVG
└── README.md                 # Documentação
```

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica.
- **CSS3**: Layout responsivo com Flexbox/Grid e variáveis para consistência.
- **JavaScript (Vanilla)**: Manipulação do DOM e navegação entre telas.

## Funcionalidades

- Navegação entre telas simulando SPA.
- Adição de tarefas (simulação visual).
- Marcação de conclusão.
- Interface mobile-first otimizada.
